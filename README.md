# DDD Tutorial For Golang
This is DDD tutorial for goland

## Usage

1. Create .env file in src/training/app and root directory

ex:
```
DB_NAME=gopherDB
DB_USER=gopher
DB_PASS=password
DB_PORT=3306
DBMS=mysql
HOST=ddd-training-mysql
TZ=Asia/Tokyo
```

2. Run `docker-compose up`

3. Go to `localhost:8080`
